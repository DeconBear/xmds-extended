当前分支增加了什么

  • 不依赖 MSYS / WSL 的 Windows 原生支持。
  • 通过 GitHub Release 发布的便携安装器。
  • 可在任意目录执行的 run_xmds。
  • 可升级、可卸载的便携安装流程。
  • 面向公开维护的 Pages、Release Notes 和说明材料。

快速安装

  1. 从 Releases 下载最新 Windows 便携 zip。
  2. 解压到临时目录。
  3. 运行 powershell -ExecutionPolicy Bypass -File .\install_windows.ps1。
  4. 选择较短的安装目录，例如 D:\xmds。
  5. 按需决定是否加入 PATH。
run_xmds C:\path\to\simulation.xmds
