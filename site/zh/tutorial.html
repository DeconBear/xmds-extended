教程

从安装、运行到理解 XMDS 脚本结构。

这份页面优先服务于 xmds-extended 的实际使用路径：先装 Windows 便携版，再跑 Lorenz 示例，然后把一个 .xmds 文件和它最终生成的输出文件对应起来。

推荐路径
第 1 步
先安装 Windows 便携版。
第 2 步
先跑通 examples/lorenz.xmds。
第 3 步
再去读官方 worked examples。

1. Windows 安装

  1. 从 Releases 下载最新便携 zip。
  2. 解压到临时目录。
  3. 运行安装脚本。
  4. 选择较短的安装路径。
  5. 按需决定是否加入 PATH。
powershell -ExecutionPolicy Bypass -File .\install_windows.ps1

2. 用 Lorenz 示例验证

安装完成后，先编译并运行仓库自带的 Lorenz 示例。

run_xmds E:\path\to\xmds-extended\examples\lorenz.xmds

运行后通常会得到 lorenz.cc、lorenz.exe、lorenz.cmd、lorenz.h5 和 lorenz.xsil。

3. 文件流水线

  • .xmds：模型描述
  • .cc：生成的 C++ 源码
  • .exe：编译后的程序
  • .cmd：补齐 DLL 路径的 Windows 启动器
  • .xsil / .h5 / .dat：输出结果

4. 先从简单脚本读起

examples/lorenz.xmds 很适合作为第一个脚本，因为它能清楚展示 XMDS 的主要结构：

  • <features>：全局常量
  • <geometry>：传播维度与空间维度
  • <vector>：演化变量
  • <sequence>：积分器与方程
  • <output>：输出量
